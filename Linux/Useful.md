<H1>Find Things</H1>

--find--  :arrow_forward: search for files in a directory hierarchy

```
sudo find -name *.txt 2>/dev/null
```

--which-- :arrow_forward: locate executable files

```
which nc
```

<h1>Which commands can I run as sudo?

```
sudo -l
```
