<H1>Find Things</H1>

__find__ :arrow_forward: search for files in a directory hierarchy

```
sudo find -name *.txt 2>/dev/null
```

__which__ :arrow_forward: locate executable files

```
which nc
```

<h1>Which commands can I run as sudo?</h1>

```
sudo -l
```
